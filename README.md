# 단국대학교 식품자원경제학과 트랙제 자가진단

2026학년도 단국대학교 식품자원경제학과 모듈형 트랙제 교육과정을 기준으로, 학생이 관심 트랙과 이수 과목을 체크하면 부족 모듈, 남은 학점, 추천 과목을 확인할 수 있는 정적 웹앱입니다.

## 배포 URL

[https://dku-track-diagnosis.vercel.app](https://dku-track-diagnosis.vercel.app)

![배포 첫 화면](docs/assets/readme-home.png)

## 사이트 목적

트랙제 신청 과정에서 학생이 직접 확인하기 어려운 내용을 한 화면에서 정리하는 것이 목표입니다.

- 내가 선택한 트랙에서 어떤 모듈이 부족한지 확인
- 이미 들은 과목과 앞으로 들어야 할 과목 구분
- 복수 트랙 선택 시 트랙별 충족/부족 상태 비교
- 학년/학기 기준으로 남은 과목 후보 확인
- 실험실 탭에서 현재 이수 과목 기준으로 달성 가능성이 높은 트랙 탐색

이 도구는 자가진단 보조용으로 제작했습니다. 자세한 최종 졸업, 트랙 인정 여부는 학과 공식 안내로 확인해야 합니다.

## 주요 기능

- 트랙제 설명, 트랙/모듈표, 자가진단, 결과, 실험실, 문의사항 탭 제공
- 5개 트랙 기준 모듈 충족 여부 계산
- 복수 트랙 선택 및 트랙별 부족 과목 분리 표시
- 보완 후보 과목을 학년/학기별로 정리
- 이수 과목 브라우저 저장
- 결과 PDF 출력 지원
- Vercel 배포용 `vercel.json` 포함

## 최근 업데이트 기록

### 2026-06-18 화면 보강 및 실험실 탭 추가

이번 업데이트는 교수님 피드백과 실제 학생 사용 흐름을 반영해, 화면을 더 짧고 한눈에 읽히도록 다듬는 데 집중했습니다.

- `설명` 탭을 첫 화면으로 유지하고, 사이트 사용법 안내 창을 단계형 흐름으로 정리했습니다.
- `자가진단` 탭의 과목 선택 방식을 학년·학기별 전공 이수 표 중심으로 바꿨습니다.
- 학년·학기 필터 버튼을 표 안에 배치해, 필요한 학기만 빠르게 좁혀 볼 수 있게 했습니다.
- `결과` 탭에는 전체 진행률, 남은 과목 수, 트랙별 부족 과목, 3학점 기준 보완 과목 수를 더 명확히 표시했습니다.
- 필수 과목 누락 영역은 밝은 붉은 계열로 강조하되, 과목명과 세부 정보는 검정 계열로 유지해 가독성을 높였습니다.
- `트랙 추천` 탭은 현재 체크한 과목 기준으로 5개 트랙의 달성 가능성을 비교하는 화면으로 정리했습니다.
- `실험실` 탭을 `도구&정보` 다음 위치에 추가했습니다. 현재 학년·학기를 입력하면 남은 정규학기, 학기당 목표 과목 수, 수강신청 우선순위를 확인할 수 있습니다.
- 실험실 탭 상단에 기능 설명 블록을 추가하고, 카드 안에서 길게 줄바꿈되던 문장을 축약해 가독성을 높였습니다.
- `도구&정보` 탭에는 학과 홈페이지, 학과 유튜브, 트랙제 안내 영상, 트랙별 모듈·과목표를 참고용으로 묶었습니다.

검증 결과:

- `pnpm run test` 기준 20개 테스트 통과
- `pnpm run build` 기준 TypeScript 검사 및 Vite 프로덕션 빌드 통과
- 로컬 브라우저에서 설명, 자가진단, 결과, 트랙 추천, 도구&정보, 실험실 탭 화면 흐름 확인

남은 리스크:

- 실제 과목 개설 학기와 폐강 여부는 수강신청 시스템 기준으로 달라질 수 있습니다.
- 트랙 인정 기준은 학과 공식 안내가 최종 기준입니다.
- 실험실 탭의 수강신청 전략은 학기당 트랙 보완 과목 2개를 기준으로 한 참고 계산입니다.
